1. Gdzie jest wstawiany?

    Wstawia się zawsze w części nagłówkowej dokumentu (pomiędzy znacznikami a )

  2. Składnia teoretyczna (w ramce, konieczne wcięcia)

    < head>
    < style type="text/css">
    selektor1 { cecha: wartość; cecha2: wartość2... }
    selektor2 { cecha: wartość; cecha2: wartość2... }
    < /style>
    < /head>

  3. Jakie mogą być selektor?

    Selektor prosty (selektor typu)

    Selektor uniwersalny

    Selektor potomka

    Selektor dziecka

    Selektor braci

    Selektor atrybutu

  4. Jakie mogą być cechy i wartości?

    Jako "cecha" (w deklaracji stylu - powyżej) należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (opisane w kolejnych rozdziałach).
    Natomiast jako wyraz "wartość" wpisujemy dokładną wartość atrybutu.
